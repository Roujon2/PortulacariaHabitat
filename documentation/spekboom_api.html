Spekboom Mapping API Documentation

Introduction

Spekboom Mapping is a web application that allows users to draw and classify polygons on a map. This API allows the app to authenticate and authorize users; and to retrieve and create polygon data.

Authentication

To authenticate, you need to include your API key in the headers of all requests.

Endpoints

GET /api/v1/users

Retrieve a list of all users.

Request:

GET /api/v1/users

Response:

{
  "users": [
    {
      "id": 1,
      "name": "John Doe",
      "email": "john@example.com"
    },
    {
      "id": 2,
      "name": "Jane Doe",
      "email": "jane@example.com"
    }
  ]
}

POST /api/v1/users

Create a new user.

Request:

POST /api/v1/users
Content-Type: application/json

{
  "name": "New User",
  "email": "newuser@example.com"
}

Response:

{
  "id": 3,
  "name": "New User",
  "email": "newuser@example.com"
}
